StoryBreeze is an original classic WordPress theme with a modern storefront-inspired design.

== Description ==

StoryBreeze is built for users who prefer a classic WordPress theme workflow (header.php/footer.php/templates) instead of Full Site Editing.

Highlights:
- Professional header and footer with stable menu behavior.
- Attractive storefront-style homepage layout.
- Optional local demo import page under Appearance > StoryBreeze Setup.
- No required plugins.
- No external assets, no tracking, no admin upsells, no activation redirects.

== Installation ==

1. Upload the `storybreeze` folder to `/wp-content/themes/`.
2. Activate StoryBreeze.
3. Go to Appearance > StoryBreeze Setup and click "Import Starter Content".

== Demo Import ==

- Location: Appearance > StoryBreeze Setup
- "Import Starter Content" creates and assigns:
  - Homepage, shop, about, services, contact, and blog pages
  - Primary and footer menus
  - Starter blog posts
- "Import Pages and Menus" imports only pages and menus.

== Accessibility ==

- Skip link to `#main`
- Keyboard-visible focus styles
- Semantic landmarks in header, main, and footer

== Privacy ==

- No remote calls for core functionality
- No telemetry
- No activation redirects
- No persistent admin nags

== Assets ==

No bundled third-party libraries, web fonts, or icon packs are included.
